## Cloud-logger

Send and save your logs in cloud.

## Code Example

```const logger = require('cloud-logger')('my-logger-server.com')```

## Motivation

Often all you need is simple flexible logger that saves everything in your own storage.

## Installation

```npm install cloud-logger -S```

## API Reference

console
warn,warning
error
cloud

## Tests

```npm run test```
